Question-1: What are the different ways to select an element in the DOM?

There are several ways to select. we can select with id, classes , css class, tag name. For example, getElementByID, getElementsByClassName etc. When we mention id, we have to keep in mind that id mean only one element

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML mean getting access of whole included html but on the other hand innerText mean access of only text thats inside the element. textContent is just like innerText but unlike innerText, textContent doesnt trim spaces

Question-3: What is event delegation in the DOM?

Event delegation is a method add a single event listener to parent element instead of multiple child

Question-4: What is event bubbling in the DOM?

Event bubbling is where the event start from target element but it also goes through parent element and only end after reaching root

Question-5: How do you create, add, and remove elements using JavaScript?

we can use document.createElement to create new element. for example document.createElement ("p") . We can use .appendChild to add element . and we can use .removeChild to remove a element
